# 📰 NLP Sentiment Analysis App

A simple Streamlit app that analyzes the sentiment of text and shows whether it is **Positive**, **Negative**, or **Neutral**.

## 🚀 Overview

This project uses Natural Language Processing (NLP) to evaluate the sentiment of user-provided text. It is built as an interactive web app using Streamlit, making it easy to test different inputs and view results instantly.

## ✨ Features

- Enter any text and analyze sentiment
- Classifies text as:
  - Positive
  - Negative
  - Neutral
- Interactive user interface with Streamlit
- Fast and easy to use

## 🛠 Tech Stack

- Python
- Streamlit
- NLP library used in the project
- Pandas / other supporting libraries if applicable
